# Security Policy

## Supported Versions

The following versions of **UAV-Architect-Global** are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of autonomous systems seriously. If you discover a vulnerability that could compromise the safety or integrity of a UAV mission, please follow these steps:

1. **Do not open a public issue.**
2. Send a detailed report to the maintainer via GitHub Private Reporting (if enabled) or contact **Bahattin Yunus Çetin** directly.
3. Include a description of the vulnerability and a proof-of-concept if possible.

We aim to acknowledge all reports within 48 hours and provide a fix or mitigation strategy as soon as possible.
